# Bricks Design System — Docs Export

## Contents
- **Bricks Docs.html** — the documentation site, fully self-contained (fonts, images, styles, and scripts are all embedded). Double-click to open in any browser; works offline. Includes the dark-mode toggle (moon icon, bottom of the left nav) and the working "Design System (v1.2.0)" download button.
- **bricks-design-system-tokens-v1.2.0.zip** — the design tokens bundle (color, spacing, typography, radius JSON + manifest), same file the in-page download button serves.
- **figma-left-nav.js** — Figma script that rebuilds the left nav as native auto-layout frames. Run it via the Scripter plugin in Figma.

## Notes
- Everything you need is in **Bricks Docs.html** — no other files or folders are required for it to work.
